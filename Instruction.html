1. Select Stressor
  2. Select Confidence interval to be calculated around the mean value of the stressor
    1. MD defaults to 80% CI
  3. Select Percentile value to calculate significance between Classifications
    1. For stressor where high values are detrimental a higher percentile is need (i.e. 0.9)
  4. Select the number of iterations to run
    1. Higher number of iterations will generate a ‘truer’ p-value
  5. Press Calculate button
